# 📝 Notes CLI App

## Description

This is a simple notes application built with Python and Textual.

You can create, update, delete, and search notes directly from the terminal using a graphical interface.

---

## Features

- Create new notes
- Edit existing notes
- Delete notes
- Search notes by title
- Simple UI in the terminal

---

## Tech Stack

- Python
- SQLite
- Textual
